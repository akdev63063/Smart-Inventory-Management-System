# Smart-Inventory-Management-System
📦 Smart Inventory Management System  A secure and scalable Inventory Management System built using Python (Flask) with JWT Authentication. This system allows businesses to manage products, stock levels, and users with secure login functionality.

🚀 Features

🔐 User Authentication using JWT (JSON Web Token)

👤 User Registration & Login

📦 Add / Update / Delete Products

📊 Track Stock Quantity

🔎 Search Products

🛡️ Protected Routes (JWT required)

💾 Database Integration (SQLite / MySQL)

🌐 REST API Based Architecture 

🛠️ Tech Stack
Technology	Usage
Python	Backend Development
Flask	Web Framework
Flask-JWT-Extended	Authentication
SQLite / MySQL	Database
HTML/CSS/JAVASCRIPT	Frontend
Git & GitHub	Version Control 

🔑 JWT Authentication Flow

User logs in with username & password

Server verifies credentials

JWT Token is generated

Token is sent to client

Client sends token in headers for protected APIs
